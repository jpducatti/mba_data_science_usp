##  ESTATÍSTICA PARA CIENCIA DE DADOS

1 - INFERENCIA BAYESIANA: quando o pesquisador tem um conhecimento que não está presente na base de dados

2 - analise exploratório: 
 2.1explorar os dados para fazer análises para descobrir aspectos e insights para identificar pistas e evidências dos dados. 
 É um trabalho de detetive

 3 - Viz para variável qualitativos/categórico: grafico de pizza, barras, tabela de frequencias, graf de mosaico
     Viz para variavel quantitativo: histograma, boxplot, tabela de frequencia com faixas

4 - Medidas de associação: medidas que relacionam duas variáveis. eg: salário está relacionado com escolaridade?

5 - Medidas de associação: essencial saber o tipo das variáveis a serem analisadas
    Covârianca e correlação para variaveis quantitativas. Analisa a variância conjunta.
    Correlação de pearson: padronização das covâriancas pelas raizes das variâncias das variaves. Fornece quão forte é a relação linear de duas variáveis quantitativas
    Correção de spearman: relação monotônica das variáveis (uma sobe a outra sobe, e vice-versa) e que não necessariamente é linear

6 - Dif grafico de barras
Histograma  = para variaveis quantitativas contínuas (as barras ficam adjacentes)
graf de barras = variaveis qualitatitivas (barras separadas)

7 - intersecção de caixas de box-plot igual visto na aula significa que os dados não indicam diferença de média estatisticamente significante nas amostras. Mas isso dá um cheiro, 
para comprovar precisa de um teste de hipótese
