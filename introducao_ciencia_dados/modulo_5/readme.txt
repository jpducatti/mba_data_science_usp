1. Erro | ruído é o que não está imbutido no modelos

2. Regressão Linear Simples: y = b0 + b1*x; Usar dados de treinamento para predizer b0 e b1 e criar o modelo, 
teste para avaliar o modelo e sua performance. Objetivo é minimizar os resíduos.

3. Como avaliar modelo: RMSE e R2

4. Avaliar se o modelo é significativo através do teste de hipótese usando distribuicao t de student

5. Regressão Linear Múltipla: o modelo é linear nos parâmetros e não nas variáveis

6. Simplificando Modelos via Regularização: quão bom são os ajustes via Simples ou Múltiplo. De forma a evitar o
overfitting, usar a Regularização. O método de RIDGE REGRESSION ou Tikhonov Regularization adiciona um termo no modelo
para controlar o overfitting.
    6.1. Outro método popular é o LASSO (least absolute shrinkage and selection Operation). O LASSO é interessante
    para fazer uma seleção de atributos
